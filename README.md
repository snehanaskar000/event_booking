# Event Management System
* # EventBook вЂ“ Event Booking & Management System

## Overview

EventBook is a modern web-based Event Booking & Management System that enables users to discover, explore, and book events seamlessly. The platform provides an intuitive user experience for attendees while offering efficient event management capabilities.

The project was developed to demonstrate practical implementation of full-stack web development concepts using PHP, MySQL, Bootstrap, and JavaScript.

## Features

### User Authentication

* User Registration
* Secure Login System
* Password Encryption using PHP Password Hashing
* Session Management
* Logout Functionality

### Event Management

* Dynamic Event Listings
* Event Details Page
* Event Categories
* Event Information Display
* Upcoming Events Showcase

### Booking System

* Event Registration
* Ticket Booking
* Booking Confirmation
* User Booking Records

### User Dashboard

* Personalized User Experience
* Booking History Tracking
* Profile Information Management

### Website Pages

* Home Page
* About Us Page
* Events Page
* Contact Page

### Security Features

* Password Hashing
* Prepared Statements (SQL Injection Prevention)
* Input Validation
* XSS Protection using htmlspecialchars()
* Secure Session Handling

### Responsive Design

* Mobile-Friendly Interface
* Bootstrap 5 Framework
* Modern UI Design
* Cross-Browser Compatibility
